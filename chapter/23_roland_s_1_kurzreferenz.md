# 23. Roland S-1 - Kurzreferenz

- **Sektionen**: OSC, Filter, AMP, ENV, LFO, EFX
- **Wichtige CCs**:
  - 74 -> Filter Cutoff
  - 71 -> Resonance
  - 73 -> Attack
  - 75 -> Decay
  - 72 -> Release
  - 91 -> Reverb Level
  - 92 -> Delay Level
- **Praxis**: EZ-AG Noten triggern Sequencer, Midi Commander CC-Steuerung
